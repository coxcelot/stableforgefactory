StableForgeFactory on Base Mainnet

Decentralized Stablecoin Factory

Overview

StableForgeFactory is a decentralized factory contract deployed on Base Mainnet that allows users to create euro-backed stablecoins (EUSD), provide liquidity via Aerodrome, and earn yield through Aave integration.
